📝 編輯個人資訊

🔗 常用連結

  • 點擊按鈕開啟 LINE
  • 點選右上方加入好友
  • 點選行動條碼
  • 點選顯示行動條碼
  • 點選複製連結並貼到上方欄位即可
  • 點擊按鈕開啟 Facebook
  • 登入後點編輯個人檔案旁邊的...
  • 滑到最下面找到複製個人檔案連結
  • 並貼到上方欄位即可

🔐 修改密碼

📘 (電腦版)如何取得 Facebook 個人頁面連結

  1. 點擊「我知道了」後，會開啟 Facebook 網站。
  2. 登入你的帳號。
  3. 點擊頁面右上方的你的名字，進入個人主頁。
  4. 複製瀏覽器網址列中的完整網址。
  5. 回到這頁，貼到「Facebook 好友連結」欄位中。
